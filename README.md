# Frontline Echo Website

GitHub Pages ready static website for Frontline Echo.

## Upload instructions

Upload the CONTENTS of this folder to the root of your GitHub repository, not the folder itself.

Your repository should show:

- index.html
- style.css
- script.js
- CNAME
- README.md
- images/
- programme/

The custom domain file should contain:

frontline-echo.uk

## Important

All photos are used exactly as uploaded. No AI editing or face alteration has been applied.
